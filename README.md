# Tea at 9 — Bubble Tea Website

Landing page für **Tea at 9 · Bubble Tea & Co.** in Stendal & Magdeburg.

## Über die Seite
- Statische Website (HTML, CSS, Vanilla JS) — keine Build-Tools nötig.
- Warmes, helles Design mit echten Produkt- und Ladenfotos.
- Scroll-Animationen via [AOS](https://michalsnik.github.io/aos/), animierte Zähler und ein Menü-Filter.
- Echte Google-Bewertungen (4,6 ★ · 134 Rezensionen).
- Vollständig responsiv (Desktop, Tablet, Mobil).

## Struktur
```
index.html      # Seiteninhalt & Sektionen
styles.css      # Design / Layout
script.js       # Nav, Menü-Filter, Warenkorb-Toast, Zähler, AOS
assets/         # Logo & Fotos (hero, Produkte, Laden)
```

## Lokal ansehen
```bash
python3 -m http.server 8000
# dann http://localhost:8000 öffnen
```
